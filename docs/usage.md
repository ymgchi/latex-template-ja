# 使い方

## 基本的な使い方

### 1. リアルタイムプレビューの起動

```bash
cd latex-template-ja
make live
```

実行すると、PDFファイル名を聞かれます：

```
==========================================
  LaTeX Live Preview
==========================================

PDFファイル名を入力 (拡張子不要、Enter=main):
```

例：`report_01` と入力すると `report_01.pdf` として保存されます。
何も入力せずEnterを押すと `main.pdf` になります。

### 2. ファイルの編集

以下のファイルを編集してください：

- `sections/title.tex` - タイトル、科目名、提出日、学籍番号、氏名、学科
- `sections/main_issue.tex` - レポート本文

### 3. 自動更新

ファイルを保存すると、自動的にPDFが更新されます。

### 4. 終了

`Ctrl+C` で終了します。

## その他のコマンド

### 手動ビルド

```bash
make pdf
```

### 一時ファイルの削除

```bash
make clean
```

### 自動ビルド（プレビューなし）

```bash
make watch
```

ファイルを保存するたびに自動でビルドされます。別のターミナルでPDFビューアーを開いてください。

## よくある質問

### Q. 日本語ファイル名は使えますか？

A. 使えます。ただし、他のOSに送る際に文字化けする可能性があるため、英数字を推奨します。

### Q. 図や表を追加したい

A. `sections/main_issue.tex` にコメントアウトされた例があります。コメントを解除して使ってください。

### Q. 参考文献を追加したい

A. `sections/main_issue.tex` の最後にある参考文献セクションのコメントを解除してください。

### Q. 再提出の場合はどうすればいい？

A. `sections/title.tex` の先頭にある再提出表記のコメントを解除してください。

## トラブルシューティング

### ビルドエラーが出る

```bash
make clean
make pdf
```

でクリーンビルドしてみてください。

### PDFが更新されない

一度 `Ctrl+C` で終了して、再度 `make live` を実行してください。
