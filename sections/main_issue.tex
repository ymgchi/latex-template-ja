\section{考察}
結果に対する考察を記述する。以下のような観点から考察するとよい。

\begin{itemize}
    \item 予想した結果と実際の結果の比較
    \item 誤差や予想と異なる結果が生じた原因
    \item 他の方法との比較
    \item 結果の応用可能性
    \item 関連する理論についての考察
\end{itemize}

注意：単なる感想や実験内容の説明、言い訳は考察ではない。
